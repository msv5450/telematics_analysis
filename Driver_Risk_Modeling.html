# this function does Multi colinearity detection -> correlation of all variables
def plot_heatmap(df, fig_size=(10, 7)):
    fig = plt.figure(figsize=fig_size)
    sns.heatmap(df.corr(), annot=True)
    plt.title('Heatmap for detecting multicollinearity', fontsize=16, color='navy')
    plt.show()
    
plot_heatmap(carTypeData)

Inference From the above heat-map we can see that there is not a high correlation between any features and hence we can continue with all of the above features to train our model. One could argue that distance shows a higher correlartion with loss but it is still much smaller than corr = 1.
