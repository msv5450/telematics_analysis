# telematics_analysis
Analyzing the telematics data from different vehicles for getting insight from the driving patterns. The conclusion from this analysis will be used in insurance industry to adjust car insurance premiums. 

The data is collected by an insurance company from the vehicles that they have insured. The PDF file describes the problem statement. The PowerPoint slides give a detailed presentation of the results.

The "GPS_telematics" code runs an analysis on the first set of data to compute velocity and accelartion of vehicles and extract driving patterns based on the dangerous driving behavior such as braking harshly.

The "Driver_risk_modeling" code does a binary classification in order to predict the risk of having accidents based on various types of vehicle data.
